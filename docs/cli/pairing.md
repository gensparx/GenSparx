# `gensparx pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/channels/pairing)

## Commands

```bash
gensparx pairing list whatsapp
gensparx pairing approve whatsapp <code> --notify
```
